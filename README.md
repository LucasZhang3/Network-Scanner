A local network scanner that uses ARP request to detect IP and MAC addresses

Requirements:
1. Python 3.x
2. pip install scapy
3. pip install dearpygui

How to Use:
1. Run the script
2. Input IP and Subnet (found via command prompt ---> ipconfig)
3. Press Scan
